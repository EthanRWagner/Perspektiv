# Perspektiv
CSC 307 Web Application

# To set up on your machine:
1. clone repo
2. run `npm install` in root, frontend and backend

# To run:
1. run `npmrun dev` in backend
2. run `npm start` in frontend

   - Linebreak: Windows
   - Quotes: double
   - Semicolon: always
